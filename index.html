I develop web apps.

Serif.nu

Serif.nu is a course scheduling application designed to help Northwestern students plan their class registration more easily and quickly. Motivated by frustrations in using Northwestern's own course registration system, this project served as my introduction to JavaScript and web development in general. The first version was released Fall 2015, and a new version, rebuilt from scratch, was released Winter 2017.

Role:

Creator (layout design, front & back end development, user testing, site maintenence)

Version 1 Learning Focus:

JavaScript & jQuery, flask, PostgreSQL

Version 2 Learning Focus:

Git, React & Redux, ES6, webpack, node & express

Serif.nu was the first place winner of the Northwestern University IEEE Project Showcase.

Serif.nu on Github

Census Reporter

Census Reporter aims to make the access of U.S. census data easier and simpler. This was my first experience diving into an established code base. Since joining the team, I've done both front and back end work, such as implementing full-text search in the Census Reporter API, designing a search results page, generating sitemaps, and making data download files more human readable.

Role:

Full stack developer

Learning Focus:

Django, PostgreSQL

I research physics.

Fermilab

At the Fermilab Center for Particle Astrophysics, I analyzed instrument testing data for the Array Camera for Optical to Near-IR Spectrophotometry (ARCONS). ARCONS is the first ground-based instrument in the optical through near-IR wavelength range based on Microwave Kinetic Inductance Detectors.

Tasks:

Operate ARCONS ultra-low temperature refrigerator

Test data collection using controlled photon source

Data analysis using Python

A portion of my time at Fermilab was sponsored by the Science Undergraduate Laboratory Internship.

CIERA

At the Center for Interdisciplinary Exploration and Research in Astrophysics (CIERA), I am currently investigating the properties of neutron stars, especially those in binary star systems, as they relate to the possibility of a potential gravitational wave source.

More Info Coming Soon

CLASSE

At the Cornell Laboratory for Accelerator-Based Sciences and Education (CLASSE), I collaborated on a dark matter search proposal. This experiment, if funded, would use missing mass detection to probe the existence of "dark photons" through the use of the Cornell Electron Storage Ring.

Tasks:

Test Cesium Iodide scintillating crystals for suitability

Arduino programming and circuit board maintenance for crystal testing

Use of oscilloscope and singal generators for crystal testing

Data analysis using Python

As a physicist, I am interested in exploring high energy particle physics and astrophysics. I am currently pursing a bachelor's degree in physics at Northwestern University and conducting research at CIERA. I currently serve as the President of Northwestern's Society of Physics Students.
