I develop web apps.

Serif.nu

Serif.nu is a course scheduling application designed to help Northwestern students plan their class registration more easily and quickly. Motivated by frustrations in using Northwestern's own course registration system, this project served as my introduction to JavaScript and web development in general. The first version was released Fall 2015, and a new version, rebuilt from scratch, was released Winter 2017.

Role:

Creator (layout design, front & back end development, user testing, site maintenence)

Version 1 Learning Focus:

JavaScript & jQuery, flask, PostgreSQL

Version 2 Learning Focus:

Git, React & Redux, ES6, webpack, node & express

Serif.nu was the first place winner of the Northwestern University IEEE Project Showcase.

Census Reporter

Census Reporter aims to make the access of U.S. census data easier and simpler. This was my first experience diving into an established code base. Since joining the team, I've done both front and back end work, such as implementing full-text search in the Census Reporter API, designing a search results page, generating sitemaps, and making data download files more human readable.

Role:

Front and back end developer

Learning Focus:

Django, PostgreSQL

JoonPark.me

This website is my portfolio website designed to showcase the various aspects of my life. The challenge was twofold: I needed a site design that would showcase my skills and reflect my personality, and I wanted to fully utilize the power of HTML and CSS. I was able to use minimal JavaScript to build this site, allowing me to focus on improving my HTML/CSS skills.

Learning Focus:

HTML5, CSS3
